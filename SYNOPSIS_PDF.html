🧬 DIAGNEX

PROJECT SYNOPSIS

AI-Assisted Medical Imaging & Clinical VQA Platform

1. Project Title

DIAGNEX: Full-Stack AI-Assisted Medical Imaging & Visual Question Answering (VQA) Portal

2. Problem Statement

Medical imaging interpretation requires specialized radiologists. A worldwide deficit in radiologist staffing leads to clinical delays. While automated AI diagnostic tools exist, typical models suffer from "hallucination errors" when presented with low-quality, blurry, or out-of-distribution (OOD) images. Diagnex solves this with the **Honest Uncertainty Safety Gate**, which detects poor input quality and suppresses unsafe diagnoses, routing users to specialist advice.

3. Project Objectives

  • Live Upload Integration: Process raw X-Rays, MRIs, and Pathology reports instantly.
  • Honest Uncertainty safety protocol: Gating diagnostics into Green, Yellow, and Red zones based on deep model confidence levels.
  • Interactive VQA Chat: Let users engage in multi-turn clinical chat based on the scan findings.
  • Digital Pathology laboratory rendering: Produce structured reference lab panels for biochemistry parameters.
  • Smart Specialist Locator: Guide high-risk patients to nearby clinical professionals automatically.

4. System Architecture & Dual-Pipeline

The system runs on a highly decoupled Full-Stack layout. The frontend uses React 19 for responsiveness. The backend uses Python FastAPI for files extraction and VQA calculations. A resilient local fallback engine runs client-side if the API server is offline, protecting critical access.

5. Gating Classification & Safety Thresholds

To protect clinical integrity, Diagnex maps diagnostics using a clear three-tier traffic light protocol:

GREEN ZONE High Certainty (85% - 100%)

AI displays full findings and enables the VQA chat panel for active clinician follow-up.

YELLOW ZONE Moderate Certainty (70% - 84%)

AI displays findings alongside clear disclaimer headers urging human clinical confirmation.

RED ZONE Low Certainty (< 70%)

AI suppresses self-diagnosis completely to prevent error. Direct doctor locator suite is initiated.

6. Underpinning Safety Algorithms

The confidence metric is estimated using four key computer-science and mathematical models:

  1. Monte Carlo Dropout (BNN): Runs multiple inference passes with active dropout layers. High variance between output distributions flags epistemic uncertainty.
  2. Shannon Entropy Gating: Computes the entropy over class probability vectors to detect distribution spread:
    H(X) = - ∑ P(x_i) log₂ P(x_i)
  3. Conformal Prediction: Ensures predictions stay within specified coverage bounds.
  4. Out-of-Distribution (OOD) Reconstruction: VAE distance gates restrict confidence scores if scans contain high noise or artifacts.

7. Expected Impact

Diagnex offers a reliable, safety-first framework for deploying deep learning in clinical fields. It shows how AI can assist diagnostic workflows while adhering to strict ethical protocols, ensuring patient safety remains the highest priority.
